# Installation

1. Install [node.js](https://nodejs.org/en/)
2. Clone this repo
3. Install dependencies

Using Yarn:

```
$ yarn install
```

Using Npm:

```
$ npm install
```

5. Start the environment

Using Yarn:

```
$ yarn start
```

Using Npm:

```
$ npm run start
```
